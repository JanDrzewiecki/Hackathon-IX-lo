## Quick Start

### Install Dependencies
```bash
pip install -r requirements.txt
```

### Run the Game
```bash
python3 main.py
```

Or use the start script:
```bash
./start_game.sh
```

## Controls
- **Arrow Keys / WASD** - Move
- **Mouse** - Aim
- **Left Click** - Shoot
- **E** - Activate speed boost
- **ESC** - Pause/Menu
